<h1 align="center">Eno Music</h1>

## 介绍

Eno Music 是一个基于 bilibili 的音乐播放器，完全改变了你在该平台上的体验。它重新构想了 bilibili 作为一个音乐平台的样子，并提供了独特而沉浸式的体验。
